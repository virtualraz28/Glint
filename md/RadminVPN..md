# Radmin VPN

Radmin VPN es un software gratuito de red privada virtual (VPN) para Windows que crea una conexión LAN virtual segura a través de internet. Permite conectar ordenadores distantes como si estuvieran en la misma red local, siendo ideal para teletrabajo, soporte técnico y jugar videojuegos multijugador en red local (LAN).

# Instalacion 

1. Para instalar Radmin VPN en un equipo, se debe descargar desde su pagina oficial

![vpn](/img/radmin2.png)

2. Al completar la descarga se debe ejecutar la instalacion. 

3. Posteriormente se debe conectar a la red deseada con la opcion de **Unirse a la red**

![vpn](/img/radmin3.png)

4. Posteriormente debemos ingresar los datos necesarios ( apoyarse del bloc de notas )
